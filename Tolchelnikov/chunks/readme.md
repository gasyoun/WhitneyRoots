# Основные принципы
1.  **Модель:** Система строится на базе модели «Смысл ⇔ Текст» И. А. Мельчука.
2.  **Источники:** Для глагольных корней и префиксов мы используем СТРОГО Уитни (Whitney). Данные из Monier-Williams не принимаются.
3.  **Терминология:** Используй термин «пандитская традиция» (pandit-style), а не «пандитарская».
4.  **Формат ответов:** Все пояснения, содержащие санскритскую диакритику, давай СТРОГО в блоках `code`, чтобы избежать искажения символов в Telegram.
